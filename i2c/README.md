# badgeKit/i2c
## SMEs: Team RareCircuits - @alpineearth45, @hcadam (Discord)
The current draft for the i2c leg of badgeKit is largely thanks to the work of the RareCircuits team's i2c implementation of SAOv2 (aka 1.69bis).  
  
We have not yet incorporated badgeKit/ID's schema into the /i2c examples, so things like IDs are subject to change; consider this a beta test for now. Examples and more to come.
