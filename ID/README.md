# badgeKit/ID
## SME Discords: @ReanimationXP, @joehacksalot 
This will function as a centralized ID schema for all badges, SAOs, minibages, and other #badgelife artifacts. We are attempting to come up with a schema that would be consistent across other schemas including BLE, i2c, USB, etc, and would contain some basic identifying information about the item, such as its' type (badge, SAO, minibadge), possibly team affiliation, and more.
More to come.
